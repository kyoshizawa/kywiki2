## 3.5　エラーコード

### (1) CAFISで使用するエラーコード

| エラーコード | 事象 | エラーの内容および原因 | エラー発生時の想定される対応 |
|---|---|---|---|
| C01 | 依託向会社コードエラー | 要求電文中の依託向会社コードがCAFISセンタに登録されていない場合。 | 仕向センタにて処理を中止し、依託向会社コードが誤っていないか確認する。 |
| C03 | 最大電文長エラー | 要求電文の電文長が規定値を超えている場合。 | 仕向センタは送信を中止し、電文長規定値を超えていないか確認する。 |
| C12 | 依託向センタビジー | 依託向センタの回線が全て使用中の場合。 | 再度電文を送信する。 |
| C13 | 依託向センタ回線切断 | 依託向センタが回線切断状態の場合。 | 処理を中止する。 |
| C14 | 依託向センタ個別終了 | 依託向センタが終了している場合。 | 処理を中止する。 |
| C15 | CAFISセンタ終了 | CAFISセンタのオンラインが終了している場合。 | 処理を中止し、オンライン開始後再度電文を送信する。 |
| C17 | エンコード不正 | 機種毎に規定されているエンコード種別を使用していない場合。 | 処理を中止し、再送する。 |
| C33 | CAFIS稼働中 | CAFISが稼働している場合。 | 再度電文を送信する。 |
| C34 | 加盟店ショッピングデータフォーマット不正 | 加盟店ショッピング電文のフォーマットが正しくない場合。 | 仕向センタにて処理を中止し、電文に誤りがないか確認する。 |
| C35 | アドバイス電文警告 | アドバイス電文に対する異常電文が発生した場合。 | 異常電文を受信し処理終了とする。 |
| C50 | タイムアウト | CAFISセンタでタイムアウトが発生した場合。 | 当該電文を再送する。 |
| C51 | 経路異常 | 中継中の回線障害で電文を受信した場合。 | 処理を中止し確認する。 |
| C53 | 仕向会社コードエラー | 仕向会社コードが誤っている場合。 | 処理を中止し確認する。 |
| C54 | 電文種別エラー | サービス対象外の電文種別を受信した場合。 | 処理を中止し確認する。 |
| C55 | 経路制御情報エラー | 経路制御情報の値が不正な場合。 | 処理を中止し確認する。 |
| C56 | 電文形式エラー | 電文形式が規定されていない場合。 | 処理を中止し確認する。 |
| C57 | 当該会社サービスなし | 当該会社でサービスを行っていない場合。 | 処理を中止し確認する。 |
| C58 | ニューメリックチェックエラー | ニューメリックチェックでエラーとなった場合。 | 処理を中止し確認する。 |
| C60 | 受信不能状態で電文受信 | 受信不能状態で電文を受信した場合。 | 再度電文を送信する。 |
