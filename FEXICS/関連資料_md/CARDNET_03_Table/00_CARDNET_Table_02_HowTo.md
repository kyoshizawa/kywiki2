# 第2章　各種様式の記入方法

## 2.1　加盟店センター接続申込書（JC－0001）

加盟店様センターが、①CARDNETセンターと新規に接続する際、または②申請内容を変更する際にご記入いただく用紙です。
ご記入・ご捺印の上、原本を①新規接続時は接続確認試験開始の2か月前、②変更時は変更希望日の2か月前までに株式会社日本カードネットワークへご提出ください。

①ご記入日＜必須＞  
　本申込書ご記入年月日をご記入ください。

②お申込者＜必須＞  
　貴社の会社名および、ご担当者様の所属部署、氏名、電話番号、およびFAX番号をご記入ください。

③新規・変更区分＜必須＞  
　新規接続時は「新規」を、申請内容変更時は「変更」を丸で囲んでください。  
　「変更」を選択の際は、変更箇所のみご記入ください。

④変更希望日  
　上記③の「新規・変更区分」で「変更」を指定した場合、変更希望日をご記入ください。

⑤貴社センターの情報  

a．センター所在地  
　貴社センターの所在地をご記入ください。  

b．センター名  
　貴社センターの名称、およびフリガナをご記入ください。  

　・センター名は、漢字、ひらがな、カタカナ、アルファベット大文字15文字以内にてご記入ください。  
　・フリガナは、カタカナ25文字以内（濁点、半濁点も1文字）でご記入ください。  

c．センター略称  
　貴社センターの略称をご記入ください。  

　カタカナ、アルファベット大文字8文字以内（濁点、半濁点も1文字）でご記入ください。  

d．センターID  
　貴社センターのIDをご記入ください。  

　・1桁目～2桁目は、特別な場合を除いて「3M」をご記入ください。  
　・3桁目～7桁目は、流通システム開発センターより取得したクレジット企業コードをご記入ください。  
　・8桁目～11桁目は、特別な場合を除いて「0000」をご記入ください。  

⑥ご希望のサービス内容および回線の種類

a．基本サービス  
　磁気ストライプによるクレジットカード取引において、加盟店様センターがご利用になるデータスイッチングサービスをご選択ください。  

　オーソリデータスイッチングをご利用いただく場合は、別途、貴社センターより各カード会社様宛に売上データを提供していただく必要があります。  

b．ICオーソリデータ取引  
　IC取引におけるオーソリデータスイッチングのご利用有無についてご選択ください。  

　IC取引に利用の際は、必ず「あり」を選択ください。  

c．IC売上データ取引  
　IC取引における売上データスイッチングのご利用有無についてご選択ください。  

d．クレジット金融データ出向  
　クレジット金融取引における金融データ送信のご利用有無についてご選択ください。  

　クレジット金融データ出向「あり」をお選びいただいた場合、下記2点に関する設定を「あり」にて登録いたします。  

　・クレジット金融オンライン精査（※1）  
　・クレジット金融オフライン精査データ（※2）  

　※1　詳細は、「CARDNET接続条件書（ダイレクト接続編）」別冊クレジット金融取引拡張仕様をご参照ください。  
　※2　詳細は、「CARDNET接続条件書（バッチ機能編）」をご参照ください。  

e．クレジット金融取引金額合計表  
　クレジット金融取引金額合計表のご利用有無についてご選択ください。  

　クレジット金融取引データをご利用の場合のみご選択ください。  

f．アクワイアラ判定代行  
　サービスのご利用有無をお選びいただき、いずれかの数字を丸で囲んでください。  

　アクワイアラ判定代行「0（なし）」をお選びいただいた場合は、貴社センターにてカードストライプの企業コード、会員番号等より、加盟店契約カード会社様の判定を行っていただく必要があります。  

g．加盟店識別単位  
　貴社センターにおける加盟店様の識別単位をお選びいただき、いずれかの数字を丸で囲んでください。  

　加盟店識別単位の詳細については、「CARDNET接続条件書（ダイレクト接続編）9.3」をご参照ください。  

h．回線種別  
　本番回線・試験回線について、それぞれの回線の種類・本数をご選択ください。  

　TCP/IP接続をご選択の場合は、本数記入欄に仕向コネクション数をご記入ください。  
　（仕向コネクション数のご選択は、下記の仕向コネクション数選定基準をご参照ください。）  

表2－1　仕向コネクション数選定基準

| 仕向コネクション数 | 1時間あたりのピークトランザクション件数 |
|---|---|
| 24本 | 2,000件～ |
| 8本 | ～約2,000件 |
| 4本 | ～約1,000件 |
| 2本 | ～約500件 |

i．暗号化方式　PIN暗号化キー（KPE）  
　PIN暗号化キー（KPE）の暗号化方式をご選択ください。  

j．暗号化方式　電文暗号化キー（KC）  
　電文暗号化キー（KC）の暗号化方式をご選択ください。  

k．暗号化方式　電文認証キー（KMAC）  
　電文認証キー（KMAC）の暗号化方式をご選択ください。  

l．カットオーバー時期  
　CARDNETセンター起動によるカットオーバー日付の更新処理を行う時間をご指定いただきます。  

　24時間表記でご記入ください。  

⑦オフライン精査  

a．精査データ出力方法  
　貴社センターがオフライン精査に使用する媒体をお選びいただき、いずれかの数字を丸で囲んでください。  

　基本サービスで、売上データスイッチングもしくは金融データ仕向にて「あり」をお選びいただいた場合のみご記入ください。  

b．マッチング処理代行  
　オフライン精査のマッチング処理代行サービスの利用有無をお選びいただき、いずれかの数字を丸で囲んでください。  

⑧業務開始日  
　本番業務の開始希望日をご記入ください。  

　業務開始日は、別途調整の上決定させていただきます。  

⑨カードネット使用欄  
　何も記入されないようお願いいたします。  

## 2.2　TCP/IP接続仕向センター側回線設定情報通知書（JC－0101）

＜※TCP/IP接続の加盟店様のみご提出いただきます＞

加盟店様センターの回線設定情報をご記入いただく用紙です。1加盟店様センターごとにご記入・ご捺印の上、原本を①新規接続時は接続確認試験開始の1ヶ月前、②変更時は変更希望日の1ヶ月前までに株式会社日本カードネットワークへご提出ください。

また、CARDNETセンター側回線設定情報につきましては、加盟店様センターがTCP/IP接続が決まり次第、別途株式会社日本カードネットワークより提示させていただきます。

①新規・変更区分＜必須＞  
　新規接続時は「新規」を、申請内容変更時には「変更」を丸で囲んでください。  
　「変更」に選択の際は、変更箇所のみご記入ください。  

②貴社回線設定ご担当者情報  
　貴社回線設定のご担当者様の情報をご記入ください。  

③貴社センター指定プロバイダ  
　貴社センターの指定するネットワークサービスプロバイダ名、およびその詳細についてご記入ください。  

④回線設定情報＜必須＞  
　貴社センターがCARDNETセンターとの接続に使用する回線の設置情報についてご記入ください。  

　（1）ホストIPアドレス  
　　貴社センターのホストIPアドレスをご記入ください。  

　（2）回線区分  
　　貴社センターが試験回線を保有しない場合は「本番／試験兼用」を○印で囲んでください。  
　　貴社センターが試験回線を保有する場合は「本番用」か「試験用」のいずれかを○印で囲んでください。  

⑤接続時使用ポート番号選択＜必須＞  
　貴社センターがCARDNETセンターとの接続に使用するポート番号についてご選択ください。使用するポート番号は2種類から選択可能です。  

⑥添付資料  
　本紙を提出いただく際、必ず別紙「2．添付資料一覧」の資料（貴社がネットワークサービスプロバイダと契約された際に、貴社がネットワークサービスプロバイダへ提出されたもののコピー）を添えて株式会社日本カードネットワークへご提出ください。  

　また、別紙「2．添付資料一覧」以外のネットワークサービスプロバイダとご契約された場合は、  
場合は、貴社がご契約時にネットワークサービスプロバイダに提出された資料一式のコピーをご提出ください。


## 2.3　加盟店情報設定依頼書（JC－0202）

貴社の基本的な情報をご記入いただく用紙です。1加盟店ごとに1枚、ご記入・ご捺印の上、原本を①新規接続時は接続確認試験開始の2か月前、②変更時は変更希望日の2か月前までに株式会社日本カードネットワークへご提出ください。

①ご記入日＜必須＞  
　本書ご記入年月日をご記入ください。  

②ご記入者欄＜必須＞  
　貴社センターのID、および名称、ご担当者様の所属部署、氏名、電話番号、およびFAX番号をご記入ください。  

③新規・追加・削除区分＜必須＞  
　新規接続時は、「新規」を、加盟店契約カード会社の追加をご希望の際は「追加」を、一部加盟店契約カード会社の削除をご希望の際は、「削除」を丸で囲んでください。  

　「追加」ご選択時のご記入方法  
　　追加をご希望される加盟店契約カード会社のみ「⑥加盟契約カード会社情報」にご記入ください。  

　「削除」ご選択時のご記入方法  
　　削除をご希望される加盟店契約カード会社のみ「⑥加盟契約カード会社情報」にご記入ください。  

④追加／削除希望日  
　追加／削除希望日をご記入ください。  

⑤加盟店情報  

a．加盟店識別コード  
　加盟店様を識別するコードをご記入ください。  

　・加盟店識別単位により、記入内容が異なります。  
　・加盟店識別単位の詳細については、「CARDNET接続条件書（ダイレクト接続編）」をご参照ください。  

b．加盟店屋号  
　加盟店様の屋号、およびフリガナをご記入ください。  

　・加盟店屋号は、漢字、ひらがな、カタカナ、アルファベット大文字15文字以内でご記入ください。  
　・フリガナは、カタカナ25文字以内（濁点、半濁点も1文字）でご記入ください。  

c．加盟店会社名  
　加盟店様の会社名をご記入ください。  

　漢字、ひらがな、カタカナ、アルファベット大文字15文字以内でご記入ください。  

⑥加盟契約カード会社情報  
　加盟店様と加盟契約のあるカード会社の会社名、および会社コード（7桁）をご記入ください。  

　・40社を超える場合は、事前にご相談ください。  
　・16社を超える場合は、複数枚ご記入ください。  

⑦デビットカード取扱情報  

a．J－Debit取扱区分  
　J－Debitの取り扱い有無をご記入ください。  

b．J－Debit加盟店コード／サブコード  
　「J－Debit取扱区分」で、「1．取扱有り」を選択された場合にご記入ください。  

c．その他デビット取扱区分  
　J－Debit以外のデビットカードサービスの取り扱い有無をご記入ください。  
　「1．取扱有り」を選択された場合は、（　）内にデビット決済情報の名称をご記入ください。  

d．その他デビット加盟店コード／サブコード  
　「その他デビット取扱区分」で、「1．取扱有り」を選択された場合にご記入ください。  

⑧通帳印字加盟店名称  

a．固定部  
　デビットカードの取り扱いがある場合に、金融機関が利用者の通帳に印字する加盟店様の名称をご記入ください。  

　・カタカナ10文字以内（濁点、半濁点も1文字）でご記入ください。使用可能な文字は、下記のとおりです。  
　・固定部のみ通帳に印字する金融機関が存在するため、固定部のみで加盟店様を特定できる内容をご記入ください。  
　・本項目の内容は、加盟店様センターからの要求電文中に加盟店名称の設定が無い場合（BIT43 “加盟店名／所在地” の先頭10桁がスペースの場合のみ使用します。  

b．任意部  
　デビットカードの取り扱いがある場合に、金融機関が利用者の通帳に印字する加盟店様の名称をご記入ください。  

　・カタカナ10文字以内（濁点、半濁点も1文字）でご記入ください。使用可能な文字は、下記のとおりです。  
　・本項目の内容は、加盟店様センターからの要求電文中に加盟店名称の設定が無い場合（BIT43 “加盟店名／所在地” の先頭10桁がスペースの場合のみ使用します。  

＜補足説明～使用可能文字＞  
　通帳印字加盟店名称にて使用可能な文字は、金融機関のATM等にて印字可能なものとする必要があるため、以下の文字に限定されます。  

| 種別 | キャラクタ |
|---|---|
| 数字 | 0123456789（半角文字） |
| 英字 | ABCDEFGHIJKLMNOPQRSTUVWXYZ（半角大文字） |
| カナ文字 | アイウエオ カキクケコ サシスセソ タチツテト ナニヌネノ ハヒフヘホ マミムメモ ヤユヨ ラリルレロ ワヲン（カタカナ半角大文字） |
| 濁点・半濁点 | ゛゜（半角文字） |
| 記号 | ￥「」。・（）－／＊＄＋（半角文字） |
| スペース | （半角文字） |

（注1）「－」（半角長音）は使用できません。「－」（マイナス）を使用してください。  
（注2）カナ小文字（ャ、ュ、ョ、ッ等）は使用できません。カナ大文字を使用してください。

## 2.4　個社使用予約域（JC－0801）

貴社接続先へのタグ番号および内容の通知は貴社にて行ってください。  

①ご記入日＜必須＞  
　本書のご記入年月日をご記入ください。  

②ご記入者＜必須＞  
　貴社センターID、および名称、ご担当者様の所属部署、氏名、電話番号およびFAX番号をご記入ください。  

③新規・変更区分＜必須＞

④開始／変更希望日  
　開始／変更希望日をご記入ください。  

⑤個社使用予約域  

a．SEQ．NO．

b．長さ

⑥カードネット使用欄

## 2.5　利用限度額管理サービス　累積利用額設定　依頼書（JC－0802）

利用限度額管理サービスにおいて、仕向応答電文上の累積利用額の設定有無をご指定いただく申請用紙です。  
ご記入・ご捺印の上、原本を業務開始の2か月前までに株式会社日本カードネットワークへご提出ください。  

①ご記入日＜必須＞  
　本書のご記入年月日をご記入ください。  

②ご記入者＜必須＞  
　貴社センターID、および名称、ご担当者様の所属部署、氏名、電話番号およびFAX番号をご記入ください。  

③新規・変更区分＜必須＞  
　新規接続時は、「新規」を、申請内容変更時は「変更」を丸で囲んでください。  

④設定／変更希望日  
　設定／変更希望日をご記入ください。  

⑤個社使用予約域  
　個社使用予約域（BIT62）のCN1タグ内の累積利用額設定の有無をご選択ください。  

　「有」を選択した場合、カードネットセンターから仕向センターへの応答電文の個社使用予約域（BIT62）のCN1タグ内にて、累積利用額が通知されます。  
　「無」を選択した場合、累積利用額はオールゼロが設定されます。
