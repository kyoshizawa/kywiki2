はじめに

本書は、株式会社日本カードネットワーク（以下「日本カードネットワーク」と略します）が開発したネットワークセンターとのセンター間接続において、ダイレクト接続される加盟店様が、日本カードネットワークとの間で締結する「接続基本契約書」に基づき、事務取扱を円滑に行うための必要事項を定めたものです。


用語の定義

この事務取扱要綱においては、下記の用語を使用します。

| 用語 | 意味 |
|---|---|
| カードネット<br>CARDNET | 株式会社日本カードネットワーク |
| カードネットセンター | CARDNETの運営するネットワークセンター |
| キャフィス<br>CAFISセンター | NTTデータ通信株式会社の運営するネットワークセンター |
| 加盟店様センター | カードネットセンターと接続している加盟店様POSシステムの処理センター |
| カード会社様センター | カードネットセンターと接続しているカード会社様の処理センター |
| 営業日 | 日本カードネットワークが営業を行う日<br>土日祝祭日、年末年始（12／30～1／3）を除く月曜日～金曜日<br>但し、カードネットセンター、およびサービステスクは、24時間365日運用 |
