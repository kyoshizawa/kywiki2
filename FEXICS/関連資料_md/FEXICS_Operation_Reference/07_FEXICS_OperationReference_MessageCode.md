# 7. メッセージとコード

FEXICS が出力するすべてのメッセージと、その説明および対処方法を示します。

---

## 7.1 システム情報

| ID | メッセージ | 説明 |
|----|-----------|------|
| I001 | サインオフ電文を受信しました。 | サインオフ電文を受信しました。 |
| I002 | カット対象日付を %s から %s に変更しました。 | カット対象日付変更通知 |
| I300 | 取消アドバイスの送信が完了しました。(指定通番=%ld) | 取消アドバイスの送信および応答の受信が完了しました。 |
| I301 | カットオーバー依頼電文の送信が完了しました。 | カットオーバー依頼電文送信が完了しました。 |
| I302 | サインオン電文の送信が完了しました。 | サインオン電文の送信および応答の受信が完了しました。応答電文のアクションコードは800です。 |
| I303 | サインオフ電文送信完了 | 応答受信完了時も表示 |
| I305 | 停止要求を発行しました | Daemon通常停止要求 |
| I306 | 強制停止要求を発行しました | 即時停止 |
| I307 | 障害回復処理を実行します | 回復開始 |
| I308 | 障害回復処理を実行しました | 回復完了 |
| I309 | サインオン状態変更 | 電文送受信なし |
| I310 | サインオフ状態変更 | 電文送受信なし |
| I800 | ライセンス期限間近 | 更新必要 |
| I801 | 経路を開放しました | 指定経路解放 |
